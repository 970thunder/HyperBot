代码解决助手 - 使用说明
=====================================

功能
----
专业的代码问题解决方案生成，使用硅基流动 API 的 DeepSeek-V3.2-Exp 模型。

使用方法
--------
发送以下格式的消息：
/代码 <问题描述>

示例
----
/代码 给定n，m求【n，m】区间的排序问题，如果是素数就从小到大输出，如果不是素数就从大到小输出

回复格式
--------
插件会自动将回复分为两段：

问题分析：
[分析内容]

代码及解决方案：
[代码和详细解释]

配置要求
--------
必须设置环境变量：
SILICONFLOW_API_KEY=your_api_key_here

特点
----
✓ 高优先级拦截（与 deepseek 插件互斥）
✓ 完整代码输出（max_tokens=4000）
✓ 智能格式化回复
✓ 详细日志记录

注意事项
--------
- 使用 /代码 命令后，不会触发原有的对话回复
- 确保设置了正确的 API 密钥
- 建议问题描述尽量详细

问题排查
--------
如果无法使用，请检查：
1. SILICONFLOW_API_KEY 环境变量是否设置
2. API 密钥是否有效
3. 查看日志中的错误信息
